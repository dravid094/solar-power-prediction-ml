# Solar Power Output Prediction using Machine Learning

## 📌 Problem Statement

Predict AC power output of a solar plant using environmental and operational parameters.

## 📊 Dataset

The dataset includes:

* Irradiation
* Module Temperature
* Ambient Temperature
* Inverter (Source Key)

## ⚙️ Approach

* Data Cleaning & Preprocessing
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Model Building using Linear Regression

## 📈 Results

* Achieved R² score of ~0.95
* Strong relationship observed between irradiation and AC power

## 📊 Key Insights

* Irradiation is the primary driver of power generation
* Higher temperatures reduce efficiency
* Different inverters show slight variation in output
